
## 关于Zen
	Zen 是禅的意思，由于写Zen之前的那段日子一直比较浮躁，所以取名Zen， 意在带着禅意写代码，平复浮躁的心绪。长期漫无目的的生活会使人麻木，迷失方向。  
  
	Zen 发布以来得到了很多JRS的支持，大家积极反馈， 如今Zen功能已经日益完善，由于实在精力有限，所以决定把Zen开源，希望可以有更多的JRS投入到Zen的开发中！
